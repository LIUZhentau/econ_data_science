1. 基础R语言#

1.1. 引言#

任何一门编程语言都需要投入大量的时间才能掌握。R-Introduction 是由R语言团队编写的一本官方操作手册。本章课程简要地涉及了当中的一些关键点。在本章教学后可以再自行仔细阅读 R-Introduction 。

1.2. 安装#

R语言是开源软件，在Windows，MacOS和Linux系统中都能运行。它的安装也十分便捷。

  • R project 是R语言的基础版本

  • Microsoft R Open 优化了底层的数值计算

本地操作界面

  • 默认界面是极简风格的 rgui

  • RStudio 是当前广受欢迎的IDE

  • 支持命令行界面 (command line interface, CLI)

Docker引擎

  • Docker 是一个开源引擎，开发者可以通过沙箱的形式打包他们的应用，它使得程序能够在一个统一的“运行环境”(Linux或Windows操作系统)中运行。在本地下载好Docker之后，rocker能够为开发者提供不同用途的各种镜像。

远程操作界面

CUHK 的网上平台 SCRP 提供了

  • RStudio

  • JupyterHub

远程服务器省去了安装步骤，并提供了一个一致而稳定的环境。

1.3. 帮助系统#

帮助系统是我们学习新语言的最重要的帮手。在R语言中，如果我们只知道函数名称但不了解它用途，我们就可以使用help(function_name)指令或者简单地用一个问号?function_name来查看。如果我们只知道当中的部分关键词，那么就可以用双问号??key_words进行模糊查询。这样它就会返回多个可能相关的函数以供选择。 例子: ?seq, ??sequence

1.4. 赋值#

<- 是最基本的赋值指令。另一种常见的赋值符号是 = 。它们的作用都是给左侧的变量附上在右侧输入的 值 。这里的 值 并非一定是数值，也可能是向量、列表或者矩阵等等形式。这些我们将会在后续提到。

# Personally I prefer "=" to "<-".
# Here are a few trivial computation and screen printing.

a <- 1
b <- 2
f = a + b # try to avoid `c`, which is an internal command
d = log(f)
e = sqrt(d)

cat("log(c) =", e, "is a simple calculation")
print(e)
cat("exp(e) =", exp(e), ". I want a nice new line. \n")
print(d)

ls() # display the objects in memory

R语言对于变量名字母大小写敏感；a 和 A 是两个完全不同的对象。

A = "abc"
cat("a is", a, ", whereas A is ", A, ".")

下面这个指令可以清空工作历史中的所有变量，规范的程序一般会把它放在整个脚本的第一行。

rm(list = ls())

1.5. 向量#

向量 是相同类型的元素的集合，比如整数(integer)、逻辑值(logical value）、实数(real number)、复数(complex number)、字符(characters) 或者 因子(factor)。在R语言中，大多数情况下我们不需要特地宣言元素的类型。

c() 可以把两个以上的向量组合为一个更长的向量。

二元运算符号 +, -, * 和 / 默认是每个元素一个一个运算的。二元逻辑符号 & | !=也是一样的。我们可以参考下面的例子来理解。

a = c(1,2,3, 4)
b = rep(c(1,2), 2)
print(a+b)

# logical vectors
logi_1 <- c(T, T, F)
logi_2 <- c(F, T, T)

logi_12 <- logi_1 & logi_2
print(logi_12)

R语言中的 因子 (factor)是一种特殊的数据结构，这与别的编程语言不同。这个数据结构在制作矩阵和数据分类中会经常用到。而 字符 (characters)则是纯文本。尽管因子往往也是字符串的形式，但是在实际使用中，二者有明显的不同。

R语言中的缺省值用NA来表示(Not Available)

a = NA; b = 3; a+b

当有一些运算不成立时，比如log(-1)，R语言会返回NAN (not a number)

log(-1)

sqrt(-1)

a = Inf
a+a

b = -Inf
a+b

1.8. 数组和矩阵#

数组 (array)是可以在两个或者以上的维度存储数据的R语言数据类型。矩阵 (matrix)是二维的数组。

  • R 以列为主顺序

  • 数组运算: 每个对应元素一个个运算

A = array(rpois(4*3*2, lambda = 1), dim = c(4,3,2)); print(A) # 3 dimensional array

B = A = array(rnorm(4*3*2), dim = c(4,3,2))
print(A+B)

处理两个长度不同的数组运算时一定要特别谨慎，这种时候特别容易遇到各种报错。

A = matrix(1:6, 3); print(A)

B = matrix(1:3, 3); print(B)

print(A+B) # produce error message

在编程中Warning和Error不同。由于长度不同导致无法运算时会报错(Error)。由于长度不同，即使可以计算的时候，程序一般也会出现Warning。请注意区分这两种情况。

b = 1:3
print(A+b)

d = 1:4
print(A+d)

  • %*%: 矩阵乘法

  • solve: 求逆矩阵

  • eigen: 求特征值和特征向量

例子: 有单一自变量 \(x\) 和常数的的OLS估计中，大学教科书中用矩阵形式将OLS表示为：

\[ \hat{\beta} = (X' X)^{-1} X'y. \]

为了在R语言中进行OLS估计，我们需要将这样的数学表达式转译成代码：

第一步: 我们需要数据 \(Y\) 和 \(X\) 来运算OLS，因此我们首先创建一个数据集。

# simulate data
rm(list = ls())
set.seed(111) # can be removed to allow the result to change

# set the parameters
n <- 100
b0 <- matrix(1, nrow = 2)

# generate the data
e <- rnorm(n)
X <- cbind(1, rnorm(n))
Y <- X %*% b0 + e

第二步: 把公式翻译成代码

# OLS estimation
bhat <- solve(t(X) %*% X, t(X) %*% Y); print(bhat)

bhat <- solve( crossprod(X), crossprod(X, Y)) %T>% print( ) # equivalent computation

第三步 (附加): 绘制散点图和回归线，进而对回归线(黑色)和真实值(红色)进行比较。

# plot
plot(y = Y, x = X[, 2], xlab = "X", ylab = "Y", main = "regression")
abline(a = bhat[1], b = bhat[2])
abline(a = b0[1], b = b0[2], col = "red")
abline(h = 0, lty = 2)
abline(v = 0, lty = 2)

第四步: 在计量经济学中，我们通常还关心假设检验的结果。最常用的就是 t 统计量。 为了检验原假设 \(H_0: \beta_2 = 1\) ，我们求解对应的 t 统计量。同样地，我们把公式翻译成代码。

\[ t = \frac{\hat{\beta}_2 - \beta_{02}}{ \hat{\sigma}_{\hat{\beta}_2} } = \frac{\hat{\beta}_2 - \beta_{02}}{ \sqrt{ \left[ (X'X)^{-1} \hat{\sigma}^2 \right]_{22} } }. \]

其中 \([\cdot]_{22}\) 是矩阵的(2,2)元素。

# calculate the t-value
bhat2 <- bhat[2] # the parameter we want to test
e_hat <- Y - X %*% bhat
sigma_hat_square <- sum(e_hat^2) / (n - 2)
Sigma_B <- solve(t(X) %*% X) * sigma_hat_square
t_value_2 <- (bhat2 - b0[2]) / sqrt(Sigma_B[2, 2])
cat("The t-statistic =", t_value_2)

1.9. 混合数据类型#

一个向量只能包含一种的元素。但是 列表 (list)可以同时装有不同类型的对象。当我们需要返回不止一种对象的时候，我们就可以使用列表。

Lst <- list(dept = "Econ", no = 5180)
Lst

Lst$dept

Lst[[2]]

例子: 当我们使用 eigen 时，我们既想知道特征值也想得到特征向量。我们就可以将其分别储存到$value和$vector中。

A = diag(2)
eigen(A)

数据框 (data.frame) 是一个用来储存数据的二维表格，就像Excel数据表一样。矩阵(matrix)同样是二维表格，但不同之处在于，它只能容纳一种元素。 现实生活中，数据可能是整数、实数、字符、分类数据等各种各样的类型。因此，在R语言当中，处理混合数据时默认使用的类型是数据框。tibble是一种新的改良过的数据框类型。

1.10. 拓展包#

R语言自带的基础包(base)十分有限，但是R有良好的统计学拓展包生态系统。这对于R语言开发者来说是宝贵的财富，其他的语言(比如Python和MATLAB)无法媲美。 大部分的R Packages都在CRAN 平台。 目前学术界的常规流程是，当统计学家发表一篇关于新的统计学方法的论文时，他们就会在CRAN平台上传对应的程序包。他们通过CRAN拓展他们的成果，开发者也能第一时间接触到最新的方法。

我们可以通过 install.packages("package_name")指令下载最新的安装包。为了调用我们需要的函数，我们可以用library(package_name)或者package_name::function_name指令在当前的程序中调用该拓展包。 前者将调用程序包中的所有函数，有时候会因为和其他函数名称相同引起冲突。后者则可以让整个环境更简洁。 Applied Econometrics with R (Christian Kleiber 和 Achim Zeileis)是一本有用的教材。 它同时也伴有程序包AER。它当中包含了常见的计量方法，比如工具变量法和稳健方差估计。

# auto install missing packages

install.packages(missing)

2.3. 自定义函数#

R有许多的内置函数。不过在实际使用中我们往往会遇到各种需要自己DIY函数的情况。 最推荐的方式是自行封装一个用户自定义函数。它有如下优点：

  1. 在搭建程序时，它可以减小代码量，使得一个复杂的程序被拆分为易于操作的各个部分；

  2. 有些脚本可能甚至有成百上千的变量，自定义函数内部的变量是局部变量而不是全局变量，这样的话它们就不容易与函数外部的变量发生混淆。自定义函数只有在输入和输出的时候才与外部程序有交互。

  3. 如果我们需要修改程序，我们只需要在函数中修改一次而不需要把每个地方进行修改。

R语言自定义函数的格式是：

function_name <- function(input) {
  expressions
  return(output)
}

例子

如果中心极限定理成立，给定样本，那么我们可以计算95%的双边置信区间

\[ \left(\hat{\mu} - \frac{1.96}{\sqrt{n}} \hat{\sigma}, \hat{\mu} + \frac{1.96}{\sqrt{n}} \hat{\sigma} \right) \]

这个的计算并不难，但是我并不知道R语言是否有内置函数能够实现这个功能。下面是一个自定义的置信区间函数：

# construct confidence interval

CI <- function(x) {
  # x is a vector of random variables

  n <- length(x)
  mu <- mean(x)
  sig <- sd(x)
  upper <- mu + 1.96 / sqrt(n) * sig
  lower <- mu - 1.96 / sqrt(n) * sig
  return(list(lower = lower, upper = upper))
}

2.4. 控制流#

控制流是所有编程语言的必备功能。 if 用于条件选择， 而 for 或者 while 用于循环。

例子

计算自由度为2的泊松分布的收敛概率。我们将实验重复1000次。

Rep <- 1000
sample_size <- 100
capture <- rep(0, Rep)

if (sample_size < 50){
      print("Sample size too small. Refuse to work")
      } else {
    for (i in 1:Rep) {
      mu <- 2
      x <- rpois(sample_size, mu)
      bounds <- CI(x)
      capture[i] <- ((bounds$lower <= mu) & (mu <= bounds$upper))
    }
    print("Asymptotic theory may work")
    cat("the emprical size = ", mean(capture)) # empirical size
}

2.5. 统计学模型#

统计学模型通常用 y~x 表示， 其中左侧的 y 是因变量。右侧的 x 为解释变量。 自带的函数是 lm 。使用方式为 lm(y~x, data = data_frame)。

R语言自带的所有回归函数都有相同的结构。学会了其中一种，自然可以举一反三。

2.5.1. 线性回归例子#

我们通过模拟数据来展示一个例子

T <- 100
p <- 1

b0 <- 1
# Generate data
x <- matrix(rnorm(T * p), T, 1)
y <- x %*% b0 + rnorm(T)

# Linear Model
result <- lm(y ~ x)
summary(result)

result 是一个包含了所有回归结果的列表。我们可以从results中看到所有的估计参数值、t检验结果、F检验结果和R方。我们可以画出真实值和拟合值来检验回归模型的拟合效果。

plot(result$fitted.values,
  col = "red", type = "l", xlab = "x", ylab = "y",
  main = "Fitted Value"
)
lines(y, col = "blue", type = "l", lty = 2)
legend("bottomleft",
  legend = c("Fitted Value", "True Value"),
  col = c("red", "blue"), lty = 1:2, cex = 0.75
)

然后我们画出拟合的直线

plot(y = y, x = x, xlab = "x", ylab = "y", main = "Fitted Line")
abline(a = result$coefficients[1], b = result$coefficients[2])
abline(a = 0, b = b0, col = "red")

legend("bottomright",
  legend = c("Fitted Line", "True Coef"),
  col = c("black", "red"), lty = c(1, 1), cex = 0.75
)

我们在此再给出另一个例子。这次我们使用数据包datasets中的女性数据来研究女性身高与体重的关系。这个数据包是R语言安装自带的，它还有其他各种类型的数据集。要查看完整的数据集列表，可以使用指令library(help = "datasets")。

# univariate
reg1 <- lm(height ~ weight, data = women)

# multivariate
reg2 <- lm(height ~ weight + I(weight^2), data = women)
# "weight^2" is a square term.
# "I()" is used to inhibit the formula operator "+"
# from being interpreted as an arithmetical one.

summary(reg1)
summary(reg2)
